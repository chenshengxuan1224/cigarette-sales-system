# 香烟零售销售系统

一个功能强大的香烟零售户每日销售管理系统，支持多维度数据统计和可视化分析。

## 功能特点

### 品牌管理
- 添加、编辑、删除香烟品牌
- 支持品牌规格、类别（高端/中端/低端）
- 入库价格、销售价格管理
- 库存管理和预警设置

### 进货管理
- 进货录入（品牌、供应商、数量、价格）
- 进货记录查询和筛选
- 自动更新库存

### 供应商管理
- 供应商信息管理
- 信用等级评级（A/B/C/D）
- 合作品牌记录

### 销售管理
- 销售录入（品牌、数量、客户类型、时段）
- 实时价格预览和利润计算
- 销售记录筛选和查询
- 自动扣减库存

### 客户管理
- 客户信息管理
- 客户类型分类（个人/商户/企业）
- 累计消费统计

### 数据统计
- 今日/本月销售数据概览
- 日期销售趋势图（柱状图）
- 利润趋势分析（折线图）
- 品牌销售排名
- 类别销售占比（饼图）
- 时段销售分析（柱状图）
- 客户类型分析（柱状图）
- 客户销售排行
- 供应商供货分析（饼图）
- 库存统计和预警
- 销售日报

## 技术栈

- HTML5 + CSS3
- JavaScript (Vanilla)
- Canvas 图表绑定
- LocalStorage 数据持久化
- Font Awesome 图标

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/your-username/cigarette-sales-system.git

# 进入项目目录
cd cigarette-sales-system

# 使用任意HTTP服务器启动
python -m http.server 8080
# 或
npx serve
```

然后在浏览器中访问 `http://localhost:8080`

## 数据导出/导入

系统支持数据的导入和导出功能，便于数据备份和迁移：
- 导出：将所有数据导出为JSON文件
- 导入：从JSON文件恢复数据

## 响应式设计

支持桌面端和移动端访问，自适应不同屏幕尺寸。
